# 家庭書摘筆記 AI 提示詞

## 使用方法
1. 把書本重點頁拍照 → Google Lens / Apple 原況文字 OCR
2. 把 OCR 文字貼到 AI（Claude / ChatGPT / Kimi）
3. 在文字**前面**貼上以下提示詞
4. AI 會輸出 Markdown 筆記，複製存成 `.md` 檔

---

## 提示詞（複製這段）

```
你是一位專業的讀書筆記整理師。請根據以下書摘內容，整理成結構化的 Markdown 筆記。

## 輸出格式要求

```markdown
# 《書名》

**作者：**
**出版年份：**
**類別：**（心理學 / 商業 / 文學 / 科學 / 自我成長 / 其他）
**閱讀者：**（誰讀的：爸爸 / 媽媽 / 姊姊 / 妹妹）
**閱讀日期：** YYYY-MM-DD

## 一句話總結
（用 1-2 句話說明這本書的核心主張）

## 核心觀點
（條列 3-7 個最重要的觀點，每個附簡短說明）

## 金句摘錄
（書中最有啟發的 3-5 句話，用引用格式）

## 實用行動清單
（讀完這本書後，可以具體做什麼？條列 3-5 項）

## 個人心得
（這裡留空給閱讀者自己填寫讀後感想）

## 推薦給誰
（這本書適合什麼樣的人讀？）
```

## 整理原則
- 保留原書的核心論點，但用自己的話重新組織
- 刪除重複或過於細節的內容
- 重點標記「可以實際應用」的部分
- 如果有數據、研究、案例，簡短保留關鍵資訊
- 金句盡量貼近原文，但可微調讓語意更清晰

以下是書摘內容：

[PASTE OCR TEXT HERE]
```

---

## 進階用法：針對不同類型的書微調

### 心理學 / 自我成長類
在提示詞中加這段：
```
額外整理：
- 書中提到的「實驗/研究」簡短說明
- 可操作的「練習/工具/框架」
- 與其他心理學概念的連結
```

### 商業 / 行銷類
在提示詞中加這段：
```
額外整理：
- 商業模式或策略的關鍵要素
- 成功案例的關鍵學習點
- 可應用於創業或職場的具體建議
```

### 文學 / 小說類
在提示詞中加這段：
```
額外整理：
- 主要角色與關係圖
- 故事主題與象徵意義
- 最打動你的場景與原因
```
